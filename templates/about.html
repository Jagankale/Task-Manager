About This To-Do App

This To-Do List application is built using Flask and Jinja2. It helps users manage tasks efficiently by organizing them with priorities, deadlines, and statuses.

The app follows a clean UI design with a light orange theme, card-based layout, and responsive design suitable for mobile and desktop devices.

  • ✔ Add, Edit, Delete tasks
  • ✔ Deadline & priority management
  • ✔ Responsive UI using Flexbox
  • ✔ Beginner-friendly Flask structure
